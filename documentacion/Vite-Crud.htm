🧪 Inicio de la Práctica: CRUD de Gestión de Tareas en React

🎯 Objetivo

Desarrollar una aplicación TODO con operaciones CRUD usando React, componentes funcionales y useState.

🔧 Paso 1: Crear el proyecto con Vite

  1. Abre una terminal y ejecuta el siguiente comando:

npm create vite@latest

  1. Cuando te pregunte:
    • Nombre del proyecto: ponle algo como crud-tareas (o el nombre que prefieras).
    • Framework: elige React.
    • Variant: elige JavaScript.
  2. Entra a la carpeta del proyecto:

cd crud-tareas

  1. Instala las dependencias:

npm install

  1. Ejecuta el servidor para confirmar que todo funciona:

npm run dev

Deberías ver el mensaje con la URL local (normalmente http://localhost:5173) y una página de inicio de Vite+React.
